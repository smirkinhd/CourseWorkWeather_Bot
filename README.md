# Телеграм-бот "Прогноз погоды"

Телеграм-бот для курсовой работы

## Архитектура 

- bot.py — управление ботом
- handlers.py — логика ответов
- services/ — бизнес-логика
- main.py — точка входа

## Библиотеки

- aiogram==3.26.0
- httpx==0.28.1

## Запуск проекта

- Установка зависимостей

´´´
pip install -r requirements.txt
´´´

- Запуск бота

´´´
python main.py
´´´
